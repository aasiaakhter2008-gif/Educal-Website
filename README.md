Educal — Modern Learning Website

Educal is a simple, clean, and responsive learning website built to help students access educational content easily. It provides a smooth user experience, organized layout, and beginner-friendly design — perfect for students and new developers.

Features

Modern and clean UI

Fully responsive layout

Easy navigation for students

Sections for courses, topics, and learning material

Fast and lightweight website

Technologies Used

HTML5

CSS3

JavaScript

Vercel (for deployment)
